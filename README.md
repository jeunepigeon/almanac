# Almanac

Neutral consumption tracker. Simple, private, no account required.

## Features

- **Track habits freely** — create substances with custom colors & icons
- **Visual calendar** — monthly grid view per substance + global overview
- **Advanced stats** — 9 metrics across 5 time windows, interactive pie chart
- **100% private** — all data stays on your phone, AES-256 encrypted backups
- **Completely free** — no ads, no paywall, no premium features
